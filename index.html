Project Initiation Form

Welcome the Boyd Waterson Asset Management National Cyclical Carpet and Paint Program project launch page. You can click the “Project Initiation Form” button to enter information on an upcoming cyclical project. Once received, the Milliken / OFS Team will reach out to the identified contact to discuss the next steps, provide an outline of the program and set up the initial Kick-Off meeting if requested. If the project is submitted via the form, it is immediately entered into the project database and during the project performance the BWAM Asst Manager will receive milestone updates to track the progress of the project.

At any time, feel free to reach out to the Program Team Lead, Melissa Heeter or the Program Executive, Mike Costas with any questions. You can forward this web page link to your associates who may be responsible for performing the cyclical project for them to initiate a cyclical project on your behalf.
